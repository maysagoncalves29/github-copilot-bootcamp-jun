# GitHub Copilot BootCamp - Edição Recife

## 🚀 Aplicação Transformada para PowerApps

Esta aplicação foi transformada de uma aplicação web JavaScript para **Microsoft PowerApps**, mantendo toda a funcionalidade original de exibição e gerenciamento de diretório de pessoas.

### 📱 Versões Disponíveis

- **Web Application** (`/src`): Versão original em HTML/CSS/JavaScript
- **PowerApps Application** (`/powerapps`): Versão PowerApps Canvas App

### 🔄 Funcionalidades Implementadas

- ✅ Exibição de lista de pessoas (Nome, Email, Telefone)
- ✅ Atualização automática de dados a cada 10 segundos  
- ✅ Botão de atualização manual
- ✅ Interface responsiva com tema rosa/azul
- ✅ Geração de dados simulados brasileiros
- ✅ Indicador de status em tempo real

### 📖 Como Usar

#### Versão Web
```bash
cd src
python3 -m http.server 8000
# Acesse http://localhost:8000
```

#### Versão PowerApps
Consulte o arquivo [`/powerapps/README.md`](./powerapps/README.md) para instruções detalhadas de importação para PowerApps Studio.

### 🎨 Design

A aplicação utiliza um tema moderno com gradientes rosa e azul bebê, proporcionando uma interface visual atrativa e profissional.
